# Доходы PDR — web/PWA версия

Это локальное web-приложение для учета доходов PDR.

## Как открыть
Откройте `index.html` в браузере.

## Как сделать почти как приложение на Android
Лучше всего выложить папку на любой HTTPS-хостинг, например:
- GitHub Pages
- Netlify
- Vercel
- свой сервер

После открытия на телефоне:
Chrome → меню ⋮ → Добавить на главный экран.

## Данные
Данные хранятся локально в браузере через localStorage.
Если очистить данные сайта/браузера, записи могут удалиться.
Периодически делайте экспорт.

## PDF
PDF-расшифровка открывается как печатная страница.
Дальше: Печать → Сохранить как PDF.
