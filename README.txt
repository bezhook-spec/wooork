Доходы PDR — исправленная PWA-версия

Файлы для GitHub Pages:
- index.html
- manifest.webmanifest
- sw.js
- icon.svg

Важно после обновления на GitHub:
1. Загрузи все 4 файла с заменой старых.
2. На телефоне открой сайт в Chrome.
3. Если старая версия всё ещё глючит: Chrome → Настройки сайта → Хранилище/Очистить данные для сайта, либо удали ярлык PWA и добавь заново.
